naked.css

super lightweight css framework

Download v0.1.2 Only 5kb minified! ♥

naked.css is a minimalistic css framework designed to get your awesome lightweight css project on the way.

Who was it built for

Certain projects demand a high performance – and you just can't afford to have a css framework with a lot of classes you won't ever use as they are. All these lines of code weigh in on the page, and end up compromising the overall performance and usability of your project.

In these cases, you may be better off writing your css code on top of something like the awesome Normalize.css. However, you may need some basic grid system, form styling, and some stuff like that. If that's your case, then naked.css was shared thinking of you. It's an attempt at the smallest set of css classes that the majority of projects might find useful – and nothing more.

It's just 5kb minified, and if you gzip it you should have it at less than 2kb.

Container

Want to wrap content on a non-fluid responsive container? This is your guy: .container. 

<div class="container">
  These are not the droids you're looking for...
</div>

Grid

The grid is a 12-column fluid grid. You need to add the .grid__column class, and then the respective modifier class. Example:

grid__column--6
grid__column--4
grid__column--2
grid__column--5
grid__column--7
grid__column--12

Background colors and height added for dramatic effect. 

<div class="grid">
  <div class="grid__column grid__column--6">
    grid__column--6
  </div>
  <div class="grid__column grid__column--4">
    grid__column--3
  </div>
  <div class="grid__column grid__column--2">
    grid__column--2
  </div>
  <div class="grid__column grid__column--5">
    grid__column--5
  </div>
  <div class="grid__column grid__column--7">
    grid__column--7
  </div>
  <div class="grid__column grid__column--12">
    grid__column--12
  </div>
  ...
</div>

Important Grid columns do not have any padding or margin. If you want to add some padding, you can add the classes .grid__column--leftpad for some left padding, and .grid__column--rightpad for right padding.

Heads up! The grid system only applies for tablet-like screens and above.

Forms

Flat styled inputs. Example: 

<form>
  <div class="form__row grid">
    <div class="form__column grid__column grid__column--6 grid__column--rightpad ">
      <label>First Name</label>
      <input type="text" value="Sherlock" />
    </div>
    <div class="form__column grid__column grid__column--6 grid__column--leftpad ">
      <label>Last Name</label>
      <input type="text" value="Holmes" />
    </div>
  </div>
  <div class="form__row">
    <label>Address</label>
    <input type="text" value="221B Baker Street" />
  </div>
  <div class="form__actions">
    <button type="submit" class="btn">Submit</button>
  </div>
</form>

Buttons

The .btn class will get you a flat button. The following modifiers are available: .btn--success, .btn--danger, .btn--large, .btn--small and .btn--block.
